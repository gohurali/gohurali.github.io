    Shell Terminal for ThreadOS (Operating System Simulator)

    In this program I created a simple terminal for an OS Simulator that runs arguments that can be delimited by a semicolon or ampersand symbol. When a semicolon delimiter is used the arguments are run sequentially, whereas when an ampersand is used, the arguments are run concurrently.
